# Job Description: Senior Frontend Architect (React / Cloud)

- **Department**: SRE & Web Architecture
- **Location**: Bengaluru / Remote
- **Experience Requirement**: 5+ Years

---

## About The Role
We are seeking an expert UI Architect to lead the modernization of our core dashboard services. You will design web structures, build reusable component libraries, and optimize client-side bundle deliveries.

## Required Technical Stack
- **Frameworks**: React, Next.js (App Router), TypeScript.
- **Styling**: Tailwind CSS, CSS3 Variables, Framer Motion.
- **State Management**: Zustand, Redux Toolkit.
- **Cloud Infrastructure**: AWS (S3, CloudFront, Route53), Docker.

## Preferred Qualifications
- Strong open-source contributions.
- Experience with WebGL/Three.js or Canvas rendering.
- Passion for optimizing web performance metrics (Core Web Vitals).
