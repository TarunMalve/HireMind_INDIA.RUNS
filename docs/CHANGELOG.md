# Changelog — HireMind Elite

All notable changes to the HireMind Elite platform are documented here.

---

## [2.0.0] - 2026-06-21
### Added
- **Landing Page V2**: Premium dark-first layout with scrolling navbar, counters, and constellation visuals.
- **Onboarding Flow**: Multi-step registration wizard for recruiters and candidates.
- **Theme Switcher**: Fluid transitions between light and dark modes with persistent local cache.
- **Explainable AI**: Formula chains showing exact multiplicative P-factor equations.
- **Candidate Verification**: MCQs based on candidate resumes; correct answers reward a verified badge.

### Changed
- **Scoring Engine**: Replaced linear weights with multiplicative scoring logic in the backend ranking service.

---

## [1.0.0] - 2026-04-12
### Added
- **Dashboard UI**: Recruiter view containing candidate lists and JD upload panels.
- **Prisma Schema**: Sourced jobs and candidate records mapped to PostgreSQL.
- **Candidate DNA**: Basic Radar charts displaying candidate competencies.
