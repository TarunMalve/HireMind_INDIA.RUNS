# Platform Features Breakdown — HireMind Elite

This document lists all features of the HireMind Elite platform categorized by user persona.

---

## 👥 1. Recruiter Features

- **Multiplicative Scoring Shortlist**: Displays candidate listings ranked by true hire probability ($P(Qualified) \times P(Available) \dots$).
- **Explainable AI Matrix**: Panel showing the exact math formula chain, P-factor breakdowns, and natural language briefs.
- **Candidate DNA Radar**: Interactive visual mapping of candidate dimensions (execution, stability, direction, etc.).
- **Hidden Gems Discoverer**: Toggle to filter out pedigree candidates and highlight individuals with matching skill adjacencies.
- **JD Upload Parser**: Drag-and-drop panel to upload job advertisements, extracting key target parameters using LLMs.
- **Future Pipeline pinning**: Allows recruiters to pin candidates to 6-18 month growth pipelines for nurturing.

---

## 🎓 2. Candidate Features

- **Career DNA Profiler**: Transparent view of how the platform ranks their profile capabilities.
- **Inline Skill Verification**: Multiple-choice challenge quizzes based on resume claims to verify credentials.
- **Verified Badge Indicator**: Unlocks a golden visual badge on candidate list profiles upon quiz verification (+30% match boost).
- **Second Chance Roadmaps**: Custom learning plans generated by AI to fill missing skill gaps.
- **Matching Opportunities Hub**: Personal dashboard displaying openings where their DNA profile matches $80\%+$.

---

## 🤖 3. System Administrator & AI Features

- **Theme Manager**: Instant, global switching between light and dark modes with persistent local cache.
- **Lucide Icon Engine**: Dynamic loading of UI vector icons based on route parameters.
- **Three.js Visuals**: Floating background constellation on the landing page representing interconnected developer talent.
- **Toast Notifications**: Interactive alert system mapping toast states (success, error, warning, info) with dynamic timing slides.
- **Mobile Hamburger Navigation**: Overlay menu enabling fully responsive interactions on smartphone and tablet screens.
